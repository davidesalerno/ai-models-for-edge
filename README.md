# AI Models for Edge    
This repo is a collection of AI models that can be used on Edge
## Models
### OVMS
- ONNX MNIST - https://github.com/onnx/models/tree/main/vision/classification/mnist
- Model Zoo Face Detection - https://docs.openvino.ai/2023.2/ovms_demo_face_detection.html
### Seldom ML Server
- LightGBM Iris - https://github.com/SeldonIO/MLServer/blob/master/docs/examples/lightgbm/README.md
- LightGBM Mushroom - https://mlserver.readthedocs.io/en/latest/examples/xgboost/README.html

#### Face Detection

Run model server with model
```
docker run -d -u $(id -u):$(id -g) --rm -v ${PWD}/ovms:/model:Z -p 9000:9000 openvino/model_server:latest --model_name face-detection --model_path /model/face_detection --port 9000 --shape auto
```

Test call

```
git clone https://github.com/openvinotoolkit/model_server.git
cd model_server/demos/face_detection/python
pip install -r ../../common/python/requirements.txt
# In case of errors remove the tensorflow-serving-api version from the ../../common/python/requirements.txt
mkdir results
python face_detection.py --batch_size 1 --width 300 --height 300 --grpc_port 9000
# Open the results folder
```

### Seldon ML Server

#### Iris
Run model server with model
```
docker run -d -u $(id -u) --rm -v ${PWD}/mlserver:/opt/models:Z -p 8080:8080 -p 8081:8081 -p 8082:8082 -ti seldonio/mlserver:1.3.5-lightgbm mlserver start /opt/models/iris
```

Test call
```
curl -s -d '{"inputs": [{"name": "predict-prob", "shape": [1, 4], "datatype": "FP32", "data": [[4.9, 3.0, 1.4, 0.2]]}]}' -H 'Content-Type: application/json' -X POST http://localhost:8080/v2/models/iris/versions/v0.1.0/infer
```
